5달동안 UI/UX 웹디자인 공부를 하며 쌓은 기록을 포트폴리오로 정리한 사이트입니다.

## 호환성

    OS X & 리눅스:

    ```sh
    npm install my-crazy-module --save
    ```


    윈도우:

    ```sh
    edit autoexec.bat
    ```

## 기술사양

HTML ver5 / CSS Ver3 / JQUERY / JAVASCRIPT / BOOTSTRAP Ver5
